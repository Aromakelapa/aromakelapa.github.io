Cara Mencari Bug Host Kuota Ketengan All Operator

Oleh Admin pada 23 Mei 2022

Oke, kemarin saya push artikel tentang cara menggunakan TCP Custom, sekarang di artikel ini saya jelaskan cara mencari bug host nya untuk dipakai di TCP Custom.

BTW, anjay btw.. tutorial kali ini berlaku untuk semua operator ya, mau Axis, Telkomsel, Indosat, Xl dan lainnya yang memiliki kuota ketengan seperti kuota Edukasi, Conference, Games, Musik, Sosmed, Video, dan macam - macam lah.

Bagaimana Caranya ?

Caranya kita menggunakan tools subdomain finder yang nantinya kita harus tau dulu kuota apa yang akan di inject, misal nih saya punya kuota Games, nah kuota Games kan pasti ngga semua Games bisa digunakan dengan kuota Games, nah kita lihat dulu game apa saja yang bisa digunakan dengan kuota tersebut. Seterusnya baru kita pilih salah satu atau semua, bisa di cek satu satu, misal di kuota games ada Nimo, Mobile Legends, PUBGM, FF dll. Baru kita gunakan tools subdomain finder untuk mencari host yang menggunakan CDN CloudFront untuk TCP Custom.

Disini saya pilih tools Subdomain Finder/Scanner yang namanya Knockpy

Apa Itu Knockpy ?

Knockpy adalah tools yang menggunakan bahasa pemrograman python3 untuk mencari/memindai subdomain dengan cepat pada domain target menggunakan metode kamus(dictionary/wordlist).

Tahap Eksekusi
  • 1. Pertama buka termux, yang belum punya download dulu dari F-Droid.
  • 2. Karna tools ini menggunakan python3 maka kita harus memasang python3 terlebih dahulu.
    • Jika kalian ada yang sudah memasang python3, lewati langkah ke 2 & 3.
    pkg install python3
  • 3. lalu pasang pip dari python3
    • pkg install python3-pip
  • 4. Lalu masukan command line dibawah, atau copy paste.
    • git clone https://github.com/guelfoweb/knock.git
    cd knock
    pip3 install -r requirements.txt
  • 5. Baru kita scan domain targetnya, yaitu nimo.tv
    • Ganti [Domain] dengan domain yang akan di scan.
    python3 knockpy.py [Domain]
  • 6. Tunggu proses scanning sampai selesai.

    • Setelah proses scanning selesai, maka akan tampil seperti di bawah.

    Yang ditandai itu (mulai dari kiri) adalah subdomain, server, realhostname dari bug kuota games nimo tv CDN CloudFlare. CDN CloudFlare pun ada di bawah tuh, biasanya bug cloudflare awalan IP Address nya 104.angkalain.angkalain.angkalain.

    Q: "Kenapa bug? emang subdomain itu bug nya apa?"
    A: entahlah, orang" menyebutnya itu adalah bug 😂
  • 6. Dan Copy lah bug subdomain atau realhostname nya, paste di aplikasi TCP Custom atau HTTP Custom (khusus dengan SSH CloudFront) untuk proses injeksi.
Penutup

Dan itulah cara mencari bug host kuota ketengan CloudFront, sekian terimakasih untuk berkunjung dan mohon maaf bila ada salah kata atau menyinggung.

Jika ada masalah error saat pemasangan python, pip, knockpy atau ada masalah lain terkait artikel, silahkan berkomentar.
